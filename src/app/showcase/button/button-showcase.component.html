The Drux Button is a versatile, easy-to-use button component designed to fit seamlessly into your web applications. With a variety of customizable options, it can adapt to different use cases, from simple actions to loading states, giving you full control over your user interface.

Available in a wide range of variants including primary, secondary, success, warning, and danger, this button ensures that you can highlight key actions with ease. Whether you need to grab attention or indicate a critical action, the Drux Button has got you covered.

In addition to standard functionality, the Drux Button also supports a loading state to visually inform users when an action is in progress, ensuring a smooth user experience. Sizes range from small to large, and you can also use the block option to make the button fill its container, providing flexibility for different layouts.

API

Here’s a list of available inputs for the Drux Button component:

Property Description
label The text displayed on the button.
type Defines the button style (e.g., primary, secondary, danger, etc.).
size The size of the button (small, medium, large).
block Makes the button take up the full width of its container.
loading Displays a loading state.
disabled Disables the button interaction.
outlined Applies an outlined style to the button.
text Makes the button a text button without a background.
raised Adds a raised effect to the button.
ariaLabel Provides an accessible name for the button.
